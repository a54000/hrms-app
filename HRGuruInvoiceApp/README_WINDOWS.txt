HR Guru Invoice App - Windows Setup

1. Install Python from https://www.python.org/downloads/
   During install, tick "Add Python to PATH".

2. Unzip this folder anywhere, for example:
   C:\HRGuruInvoiceApp

3. Double-click:
   start_app.bat

4. Open this URL in your browser:
   http://127.0.0.1:5055

The app will create invoice PDFs, MIS export, saved companies, and saved references in this folder as you use it.
